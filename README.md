# paginagamehub
Este proyecto es un sitio web de un torneo solidario de videojuegos desarrollada con HTML5, CSS y un poco de JavaScript.
## Descripción
Se trata de una página web informativa donde se presentan los detalles del torneo, las fechas, objetivos, ubicación, participantes y opciones de colaboración e inscripción.
## Tecnologías utilizadas
- HTML
- CSS
- JavaScript (básico)
## Funcionalidades
- Presentación del torneo
- Información de los videojuegos y participantes
- Sección de fechas y objetivos
- Formulario de colaboración
